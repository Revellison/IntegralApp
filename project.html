"Integral" — это прикладное программное обеспечение, созданное на языке Python с использованием графического интерфейса PyQt6. Проект демонстрирует, как искусственный интеллект может стать полноценным помощником в разработке ПО. Целью проекта было создание функционального приложения с привлечением инструментов ИИ, таких как ChatGPT, Grok и Copilot.
Синтаксис построения графиков
Линейные функции
2*x + 3 # Прямая линия
-x + 5 # Отрицательный угловой коэффициент

Квадратичные функции
x**2 # Простая парабола
-x**2 + 4*x + 1 # Парабола с линейным членом
Степенные функции
x**3 # Кубическая функция
-x**3 + 3*x**2 # Кубическая функция с доп. членами
x**0.5 # Квадратный корень
x**(1/3) # Кубический корень
